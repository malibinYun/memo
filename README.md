## Develop Environment

- Complier - **Android Studio**
- Language - **Kotlin**
- Compile SDK Version - **29**
- Optimized Device - **Galaxy s8+**
- Mininum SDK Version - **21**

## Main Used
 - Architecture - **MVVM** (Google Clean Archiecture MVVM)
	+ **Data Binding**
	+ **LiveData**
	+ **ViewModel**
	+ **Room**
- DI
	+ **Koin**
- Test Code
  * **Junit 4**
  * **Mockito**
  * **Mockito to Kotlin**
